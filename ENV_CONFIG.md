# Frontend Environment Configuration

Create a `.env` file in the Frontend directory with these values:

```
VITE_API_BASE_URL=http://localhost:8000
VITE_AGORA_APP_ID=your-agora-app-id-here
VITE_WHITEBOARD_APP_ID=your-whiteboard-app-id-here
```

## Variable Descriptions

- **VITE_API_BASE_URL**: Backend API URL (default: http://localhost:8000)
- **VITE_AGORA_APP_ID**: Your Agora App ID from https://console.agora.io/
- **VITE_WHITEBOARD_APP_ID**: Your Whiteboard App ID (optional, mostly handled by backend)
